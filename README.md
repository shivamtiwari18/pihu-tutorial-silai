# Pihu Tutorial Website
This repository contains a professional, responsive tailoring and alteration website.

## Setup Instructions
1. Download this folder.
2. Upload `index.html` to your GitHub repository.
3. Enable **GitHub Pages** in your repository settings (Settings > Pages) to host it for free.

## Design
- **Theme**: Premium neutral/beige aesthetic.
- **Framework**: Tailwind CSS.
- **Font**: Playfair Display (Serif) & Inter (Sans-serif).
